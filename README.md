# Quote That

Quote That is a mobile app for saving and sharing funny quotes with friends in private groups.

## Tech Stack
- React Native
- Expo
- TypeScript
- Supabase
- Zustand
- React Navigation

## MVP Scope
- Sign up / login
- Create group
- Join group
- Add quote
- View quotes in a group

## Run locally
1. Install dependencies
2. Create `.env`
3. Run:

```bash
npm install
npx expo start
